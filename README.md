# Modulo SumUp per Magento 2

Modulo di pagamento SumUp per Magento 2 sviluppato da SoluzioniPHP.

Il modulo integra SumUp Hosted Checkout all'interno del processo di acquisto Magento consentendo ai clienti di effettuare pagamenti online tramite:

- Carte di credito e debito
- Apple Pay
- Google Pay

## Funzionalità

- Integrazione SumUp Hosted Checkout
- Configurazione da pannello amministrativo Magento
- API Key configurabile
- Merchant Code configurabile
- Redirect sicuro verso SumUp
- Compatibile con Magento 2.4.x
- Supporto PHP 8.x
- Supporto EUR

## Requisiti

- Magento Open Source 2.4.x
- PHP 8.1 o superiore
- Account SumUp attivo
- API Key SumUp
- Merchant Code SumUp
